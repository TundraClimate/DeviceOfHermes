BattleMapChanger

戦闘中のマップを変更します。

MapManagerをComponentとして保持しているGameObject、または背景・床面のSpriteを指定してマップを適用することが出来ます。
AudioClipを指定してBGMを指定することも可能です。

public override void OnWaveStart()
{
    BattleMapChanger.SetMap(/* BackGround */, /* Floor */, /* BGM */, [ /* Ids */ ], /* Dialog color */);
}

Specified usage

SetMap(MapManager, AudioClip?)

BattleMapChanger.SetMap(map, bgm);

MapManagerとそれに紐づくGameObjectからマップを設定します。

  • map: AddComponent済みのMapManagerです。
  • bgm: BGMに使用されるAudioClipです。指定しなかった場合はデフォルトのBGMが使われます。

SetMap(Sprite, Sprite, AudioClip?, List<string>?, Color?)

BattleMapChanger.SetMap(bg, flr, bgm, dids, dcolr);

背景・底面の画像を指定してマップを設定します。
第4引数はダイアログそのものではなくTextDataModelのidです。

  • bg: 背景画像用のスプライトです。
  • flr: 底面画像用のスプライトです。
  • bgm: BGMに使用されるAudioClipです。指定しなかった場合はデフォルトのBGMが使われます。
  • dids: 表示されるダイアログが紐付いたIdのリストです。
  • dcolr: ダイアログのメインカラーです。
