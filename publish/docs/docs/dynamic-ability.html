DynamicAbility

コマンドからページ・ダイス効果を生成します。

例:

<Script>dohc_DynamicCard_uc(draw1light1)_desc([使用時] 光を1回復。ページを1枚引く)</Script>
Script="dohd_DynamicDice_sa(draw1)_desc([的中] ページを1枚引く)"

Specified usage

コマンドは"_"で区切った複数の要素で構成されています。

第一区間はDOHCもしくはDOHDのみをコマンドとして認識し、この2種類以外が指定された場合は通常のScriptとして処理されます。
DOHCはDiceCardSelfAbilityに、DOHDはDiceCardAbility用のコマンドです。
この文字列は大文字・小文字を区別しません。dohcやdohdもコマンドとして使用可能です。

第二区間はアビリティ名を指定します。これは既存のものを指定するのではなく、新規かつ他と重複しない名称である必要があります。
これはIDとしての意味を持ち、LocalizationManager等で指定するときにはこのIDを指定することでターゲットして指定することが出来ます。

第三区間以降は実際の処理を組み上げるコマンドになっています。
コマンドは現在、

  • uc(): OnUseCard
  • sb(): OnStartBattle
  • sp(): OnStartParrying
  • so(): OnStartOneSideAction
  • wp(): OnWinParrying/OnWinParryingAtk|Def
  • lp(): OnLoseParrying
  • brd(): BeforeRollDice
  • sa(): SucceedAttack
  • rd(): RollDice
  • desc(): Desc

これらに対応しており、()の中の文字列をそれぞれ指定することで処理を追加することが出来ます。

コマンドに指定する文字列は以下の値の組み合わせです。ただしdesc()のみは例外で、文字列をそのまま指定する方法になります。

  • light{N}: 光を回復します。{N}は回復する光の値です。(例: light2)
  • draw{N}: ページを引きます。{N}は引くページの枚数です。(例: draw1)
  • gain{B}{N}: KeywordBufを獲得します。{B}はKeywordBufで、{N}は獲得する値です。(例: gainstrength1)
  • ngain{B}{N}: 次の幕にKeywordBufを獲得します。{B}はKeywordBufで、{N}は獲得する値です。(例: gainstrength1)
  • nngain{B}{N}: 次の次の幕にKeywordBufを獲得します。{B}はKeywordBufで、{N}は獲得する値です。(例: gainstrength1)
  • inf{B}{N}: KeywordBufを付与します。{B}はKeywordBufで、{N}は獲得する値です。(例: infstrength1)
  • ninf{B}{N}: 次の幕にKeywordBufを付与します。{B}はKeywordBufで、{N}は獲得する値です。(例: ninfstrength1)
  • nninf{B}{N}: 次の次の幕にKeywordBufを付与します。{B}はKeywordBufで、{N}は獲得する値です。(例: nninfstrength1)
  • heal{N}: 体力を回復します。{N}は回復する値です。(例: heal99)
  • bheal{N}: 混乱抵抗値を回復します。{N}は回復する値です。(例: bheal99)
  • admg{N}: DiceStatBonusにダメージ量を追加します。{N}はボーナス量です。(例: admg1)
  • abdmg{N}: DiceStatBonusに混乱ダメージ量を追加します。{N}はボーナス量です。(例: abdmg1)
  • apow{N}: DiceStatBonusに威力を追加します。{N}はボーナス量です。(例: apow1)
  • adr{N}: DiceStatBonusにダメージレートを追加します。{N}はボーナス量です。(例: adr10)
  • abdr{N}: DiceStatBonusに混乱ダメージレートを追加します。{N}はボーナス量です。(例: abdr10)
  • amin{N}: DiceStatBonusに最低値を追加します。{N}はボーナス量です。(例: amin9)
  • amax{N}: DiceStatBonusに最大値を追加します。{N}はボーナス量です。(例: amax10)
