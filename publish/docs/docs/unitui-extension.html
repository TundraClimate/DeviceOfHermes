UnitUIExtension

キャラクターに関する表示を扱うクラスです。

例:

owner.Say("は...");

owner.view.Say("ルール違反だ！", 1f);

Specified usage

owner.Say(dlg);

通常と同じようにセリフを喋らせます。

  • owner: 喋らせるキャラクターユニットです。
  • dlg: キャラクターが喋るテキストです。
owner.view.Say(dlg, duration, overhead, scale);

キャラクターの頭上にテキストを表示します。

  • owner: 表示する対象のユニットです。
  • dlg: 表示するテキストです。
  • duration: フェードイン・アウトを除いた表示時間です。
  • overhead: テキストを表示する高さです。デフォルトは3.2fです。
  • scale: テキストのサイズ比です。デフォルトは0.7fです。
