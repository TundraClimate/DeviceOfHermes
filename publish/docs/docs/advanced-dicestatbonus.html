using DeviceOfHermes;

名前空間DeviceOfHermesに存在するクラスです。
この記事はこれを前提に書かれています。

AdvancedDiceStatBonus

既存のDiceStatBonusに追加で要素を追加した拡張版です。

dice.ApplyDiceStatBonus(new AdvancedDiceStatBonus()
{
    // 威力をX倍に受ける
    powerMultiplier = 500,

    // 威力の効果がX%増加
    powerRate = 25000,

    // KeywordBufの付与時に適用される変更
    // 例: バフの数値付与量+1000
    kwdBufModifier = (origin, ref buf, target) => buf.stack += 1000,

    // 威力が無効になる
    ignorePower = true,

    // ダイスロールの出目が最大値に近くなる重みを加算する
    // 初期値: 100
    // 合計で400-500程度になるとほぼ最大値が出るようになる
    highrollGlobalWeight = 400
});

Specified usage

powerMultiplier

powerMultiplier = mul,

威力の影響をX倍に受けます。
複数回適用した場合、それぞれ乗算されます。

  • mul: 倍率です。

powerRate

powerRate = rate,

威力の影響をX%増加します。
複数回適用した場合、それぞれ加算されます。

  • rate: 加算割合です。

kwdBufModifier

kwdBufModifier = (origin, ref buf, target) => { },

KeywordBufを付与した場合に適用される変更です。
bufは再代入可能であり、代入結果が実際に適用されるバフ効果になります。
nullを代入した場合対象のバフを付与する処理はスキップされます。

  • origin: 変更元のバフ効果です。
  • buf: 実際に使用されるバフ効果です。
  • target: バフを付与する対象です。

ignorePower

ignorePower = true

威力が無効になります。

highrollGlobalWeight

highrollGlobalWeight = weight;

ダイスロールの出目が最大値に近くなる重みを加算します。

基礎値が100で、通常の確率です。
400-500付近でほとんど最大値が出るようになります。
逆に0-99は最低値が出やすくなります。

  • weight: 重みの加算数値です。負の値も使えます。
