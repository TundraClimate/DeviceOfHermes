BattleTickAction

毎Tick実行される処理を管理します。
処理速度により変動しますが、毎秒数十回は呼び出されることになるため気をつけるべきでしょう。

例:

BattleTickAction.OnTick += () => { Hermes.Say("BOO!") };

Specified usage

OnTick

BattleTickAction.OnTick += () => { };

毎Tickの処理を追加します。
